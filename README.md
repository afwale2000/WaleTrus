# 🌱 AgriBlock Carbon Protocol (ACP)

AgriBlock Carbon Protocol (ACP) is a Celo-native Regenerative Finance (ReFi) platform that enables smallholder farmers to measure, verify, certify, and monetize sustainable agricultural practices through AI-powered carbon assessments and blockchain-based certification.

## Features

- 🌾 Farmer onboarding
- 🤖 AI carbon assessment
- 📜 On-chain carbon certificates
- 💰 cUSD rewards on Celo
- 🛒 Carbon credit marketplace
- 🔍 Public certificate verification
- 📊 Impact dashboard

## Technology

- React + TypeScript
- NestJS
- PostgreSQL
- Solidity
- Celo
- Gemini API
- IPFS

## Vision

To create transparent, trusted, and inclusive carbon markets that empower African farmers while accelerating climate action.
